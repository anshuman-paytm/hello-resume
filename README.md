# Resume Builder

A modern, dynamic resume builder built with React. Create professional resumes by simply filling out forms - no backend required!

## Features

- 📝 **Dynamic Resume Creation** - Fill out forms and see your resume update in real-time
- 🎨 **Beautiful UI** - Modern, clean interface inspired by Overleaf
- 📄 **PDF Export** - Export your resume as a PDF with one click
- 📱 **Responsive Design** - Works on desktop and mobile devices
- ⚡ **Fast & Lightweight** - Built with Vite for optimal performance
- 🎯 **No Backend Required** - Everything runs in your browser

## Sections Included

- Personal Information
- Professional Summary
- Work Experience
- Education
- Skills
- Projects
- Certifications

## Getting Started

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Usage

1. Fill out the forms on the left side for each section
2. Watch your resume update in real-time on the right side
3. Click "Export PDF" to download your resume

## Technologies Used

- React 18
- Vite
- jsPDF (for PDF export)
- html2canvas (for converting HTML to image)
- React Icons
